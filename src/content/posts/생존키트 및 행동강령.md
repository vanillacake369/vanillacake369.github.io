# Why? 왜 배움?

---

호국유공 예우는 개나 줘버린 나라에서 실제로 전쟁이 터지면 일단 살아야한다
애국심 강해 전투는 나가겠지만, 그것도 장담은 못 한다
또한 해외여행에서 불변을 당할 수 있다
따라서 개인은 개인이 생존하고 보호해야한다
이에 따라 키트와 행동강령을 정리해보았다.


# Packed 챙겨갈 것들 🎒

## 생존 키트

예비군 기동대 지금 화기/보호구는 다음과 같다
그냥 k2/m1 과 90발 주어지고 아무것도 없다
방탄도, 보조무기도 없다

| 구분 | 현역 | 동원예비군 (상비급) | 향토예비군 (지역 방어) |
| --- | --- | --- | --- |
| **화기** | K2 | K2 | M16A1 |
| 근접 | K2 + 권총(K5) | 없음 | 없음 |
| **보호구** | K150/K200 방탄복 + KPS 헬맷 | 전투조끼 + KPS ACH 헬멧 (10~20%) | 탄띠 + M1 철모 (90%+) |
| **탄약** | 5.56mm 90발 | 5.56mm 90발 | 5.56mm 90발 |
| **기타** | 수류탄 1개, 무전기(LMR) | 수류탄 1개, 무전기(LMR) | 수류탄(제한), 농기구(도끼 등) |
| 작전/임무 | 속한 부대 작전 수행 | 현역 부대로 파병해서 보강 | 지역 방어 |


따라서 평소에는 EDC 키트를 들고 다니며 우발상황을 대처하고
실제 전시 상황에는 경보 시 → 현금과 완전 군장 키트를 챙겨서 대피소로 이동, 소집 → 
내꺼 하나, 아내꺼 하나로 구성한다

- **평시**: 출퇴근 시 자동 휴대, 회사 책상에도 백업.
- **경보 시**: 현금 챙겨 집→대피소 이동, 라이프스트로로 물 확보.
- **소집 시**: EDC 그대로 들고 지정장소로. 90발 보충용 낚시줄·부싯돌 활용.

# Action 🏃‍♀️🏃‍♂️

---

무기 사용 시 근거 확보(feat.무기 활성화하자마자 영상 촬영) → 사실상 과잉방지로 실형 받을 가능성이 있으므로 정당방위로 무죄는 거의 불가능에 가깝다. 일단 폰으로 카메라를 키거나 CCTV 가 있는 곳에서 적합한 무기로 대처
대피소 이동(근처 대피소가 어디?) → 그냥 근처 지하철, 없다면 네이버맵을 빠르게 킨다
야외 취침 시 침입자 trip wire alert → 캠핑할 때는 꼭 필요하지 않다고는 말하는데 뭉거노 영상을 볼 때 캠핑박지가 아닌 아무도 없는 산 한 가운데에서는 해야할 것 같다
